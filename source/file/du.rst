du
=====================================

简介
^^^^
检测文件大小

选项
^^^^

* -a 检测所有文件
* -b 以byte为单位
* -c 显示total栏，目录总大小
* -D/H 不参考链接
* -h 可读性显示
* -k 以kb为单位
* -l 硬链接算多次
* -m 以mb为单位
* -s 只显示个总数
* -x 跳过不在一个文件系统的目录
* -d 最大检测深度
* --time=atime,ctime,mtime,status 显示最后修改时间

示例
^^^^

检测两层子目录文件或者目录大小::

    du -h -d2 /path/
